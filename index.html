Example One: Toggling classes on DOM event:

Click the button below to reveal a hidden paragraph.



When a user clicks on the button, JS fires an event that looks for an element with an id of js-reveal, finds that element, and removes a pre-existing hide class from it (The hide class adds a style property of display: none;). The JS also updates the button text. The button acts as a toggle, allowing the user to hide/show the element at will.

Example Two: Setting new CSS properties:

The following <li>s are styled on page load with hex values found in their data attributes. JS collects an array of DOM elements (<li>s in this case) by looking for js-list-item classes, gets the color values from their data-color attributes, and applies the color: #hex; values to the DOM elements in the array.

These styles do NOT originate from an external .css file.

  • RED
  • ORANGE
  • YELLOW
  • GREEN
  • BLUE
  • INDIGO
  • VIOLET
This list does NOT have js-list-item classes or data-color attributes, so the JavaScript logic ignores it.
